## 🔧 輔助工具 Skills（6個）

| Skill | 功能 | 使用時機 |
|-------|------|----------|
| **prompt-extractor** | 從海量提示詞提取可複用模組 | 學習新風格時 |
| **prompt-xray** | 逆向工程，提取「如何做X」知識 | 分析優秀提示詞 |
| **universal-learner** | 自動學習並存入資料庫 | 持續積累知識 |
| **prompt-analyzer** | 分析、對比、推薦提示詞 | 查看已生成結果 |
| **domain-classifier** | 智能判斷領域歸屬 | 自動路由決策 |
| **prompt-generator** | 基礎提示詞生成 | 簡單場景快速生成 |

## 🎯 快速選擇指南

```
你的需求是什麼？
       │
       ├─ 有人物嗎？
       │     │
       │     ├─ YES → intelligent-prompt-generator
       │     │         （除非強調藝術技法）
       │     │
       │     └─ NO → 主體是什麼？
       │              │
       │              ├─ 風景/靜物/藝術繪畫 → art-master
       │              ├─ 產品/商品/電商圖 → product-master
       │              ├─ 海報/PPT/UI設計 → design-master
       │              └─ 視頻/動畫/運鏡 → video-master
       │
       └─ 需要完整設計系統（模板）？
             │
             ├─ Apple 風格 → design_templates (apple_*)
             ├─ Google 風格 → Material Design 模板
             ├─ 微軟風格 → Fluent Design 模板
             └─ 手繪風格 → 手繪信息圖模板
```

## 📎 附錄：模板查詢方式

### 查詢所有模板

```python
import sqlite3
conn = sqlite3.connect('extracted_results/elements.db')
cursor = conn.execute('SELECT template_id, chinese_name, usage_scenarios FROM design_templates')
for row in cursor:
    print(row)
conn.close()
```

### 查詢特定領域元素

```python
cursor = conn.execute('''
    SELECT element_id, name, ai_prompt_template
    FROM elements
    WHERE domain_id = 'design'
    ORDER BY reusability_score DESC
    LIMIT 20
''')
```
