# 現代簡約風格 - 設計原則

## 風格定位

**核心情緒**：專業、高效、剋制、現代
**目標受眾**：成人、專業人士、商務人士
**適用場景**：科技產品、商務演示、企業宣傳、專業服務

## 設計原則

### 1. 配色原則

**主色調選擇**：
- 使用深色系或中性色（Dark or Neutral）
- 飽和度可高可低（根據子風格）
- 明度適中（避免過亮或過暗）

**常用色系**：
- 深藍色系（Deep Blue）：#1F2937, #374151, #4B5563
- 紫色系（Purple）：#6366F1, #8B5CF6, #A78BFA
- 深綠色系（Dark Green）：#059669, #10B981, #34D399
- 中性灰系（Neutral Gray）：#1F2937, #374151, #6B7280

**配色規則**：
- 主色：深色或飽和色
- 輔色：與主色和諧（同色系或互補色）
- 背景：純白#FFFFFF或極淺灰#F9FAFB
- 文字：深灰#1F2937（高對比）

### 2. 形狀原則

**圓角設計**：
- 使用小到中等圓角（4-12px）
- 圓角營造現代感但不失專業
- 統一圓角規格（通常8px）

**避免**：
- ❌ 大圓角（>16px，顯得可愛不專業）
- ❌ 完全直角（0px，顯得過時）
- ❌ 圓形（過於柔和）
- ✅ 剋制的圓角，幾何感

### 3. 陰影原則

**細微陰影**：
- 使用非常軟、非常細的陰影
- 格式：`0 Xpx Ypx rgba(R,G,B,opacity)`
- X值：0-4px（水平擴散小）
- Y值：2-8px（垂直擴散小）
- 透明度：0.04-0.08（非常微妙）

**效果**：
- 微妙的層次感
- 不明顯的深度
- 保持平面化、剋制

### 4. 字型原則

**字型選擇**：
- 使用幾何無襯線字型（Geometric Sans-serif）
- 例如：Inter, SF Pro, Roboto, system-ui

**字重**：
- 標題：600-700（Semi-Bold）
- 正文：400-500（Regular-Medium）

**行距**：
- 1.5-1.6倍（緊湊但不擁擠）
- 比溫馨風格更緊湊

**字間距**：
- -0.01-0.02em（標準或稍緊）
- 專業、高效的感覺

### 5. 裝飾原則

**裝飾元素**：
- 幾何線條（直線、折線）
- 點陣（dot grid）
- 漸變色塊（subtle gradients）
- 圖示（線性圖示，outline style）

**使用規則**：
- 選擇1-2個元素組合
- 透明度：0.05-0.10（非常微妙）
- 旋轉角度：0°或45°、90°（幾何規律）
- 對齊網格（grid-aligned）

**避免**：
- 過度裝飾（>5%面積）
- 不規則形狀（organic shapes）
- emoji、手繪風格
- 過於花哨的效果

### 6. 留白原則

**留白比例**：
- 整體留白：50-55%的畫布
- 內容密集度：中等
- 元素間距：16-24px（8px的倍數）

**作用**：
- 營造專業、高效的感覺
- 讓資訊層次清晰
- 符合成人認知（資訊密度適中）

### 7. 互動原則

**按鈕/可點選元素**：
- 微妙的視覺反饋
- Hover狀態：顏色加深5-10%
- 陰影增強：0 4px 12px（但仍剋制）
- 過渡動畫：150-200ms（快速）

**專業文案**：
- "瞭解更多"
- "立即開始"
- "檢視詳情"
- "聯絡我們"

## 視覺層次

### 主要元素（最突出）
- 大標題（字號大、字重重）
- 核心資料/內容
- 主要CTA

### 次要元素
- 副標題、標籤
- 說明文字
- 圖表、圖示

### 輔助元素
- 日期、簽名
- 背景裝飾
- 細節說明

## 情緒傳遞

**透過設計傳達**：
- 專業感（剋制、幾何）
- 信任感（穩定、一致）
- 現代感（簡潔、高效）
- 高階感（留白、細節）

## 避免的陷阱

❌ 過於活潑（大圓角、鮮豔色）
❌ 過於裝飾（>10%裝飾面積）
❌ 過於立體（強陰影、3D效果）
❌ 過於複雜（資訊過載）
❌ 情緒化色彩（全粉、全綵虹）
❌ 手繪風格（不專業）

## 示例組合

**典型現代簡約風配色**：
```
主色：#6366F1（靛藍）
輔色：#8B5CF6（紫色）
背景：#FFFFFF（純白）
文字：#1F2937（深灰）
裝飾：幾何線條 opacity:0.08
```

**典型邊框陰影**：
```
border-radius: 8px
box-shadow: 0 2px 8px rgba(0,0,0,0.06)
```

## 子風格

### 1. 科技未來（Tech Future）
- 主色：深藍、紫色、青色
- 裝飾：漸變、點陣
- 用途：科技產品、AI、軟體

### 2. 專業商務（Business Pro）
- 主色：深藍、深綠、中性灰
- 裝飾：幾何線條
- 用途：企業服務、B2B、諮詢

### 3. 高階黑金（Premium Black-Gold）
- 主色：黑色#1A1A1A + 金色#D4AF37
- 裝飾：細線、漸變
- 用途：奢侈品、高階服務
