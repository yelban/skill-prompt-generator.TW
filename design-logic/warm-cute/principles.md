# 溫馨可愛風格 - 設計原則

## 風格定位

**核心情緒**：溫暖、柔軟、鼓勵、親和
**目標受眾**：兒童（3-8歲）、家長、教育工作者
**適用場景**：兒童教育、親子內容、啟蒙學習、家庭活動

## 設計原則

### 1. 配色原則

**主色調選擇**：
- 使用粉彩色系（Pastel Colors）
- 飽和度控制在60-70%（柔和但不灰暗）
- 明度在75-85%之間（明亮但不刺眼）

**常用色系**：
- 珊瑚粉色系（Coral Pink）：#FF9AA2, #FFB7B2, #FFDAC1
- 天空藍色系（Sky Blue）：#B5EAD7, #C7CEEA, #E0BBE4
- 薄荷綠色系（Mint Green）：#C7CEEA, #98FF98, #B5EAD7
- 奶油色系（Cream）：#FFFEF9, #FFF8E7, #FFEDD8

**配色規則**：
- 主色：粉彩色系中的一種
- 輔色：與主色和諧的顏色（同色系或類比色）
- 背景：奶油白或極淺的粉彩
- 文字：暖炭灰#2C2C2C（避免純黑）

### 2. 形狀原則

**圓角設計**：
- 所有元素使用大圓角（16-24px）
- 圓角營造柔軟、友好、無威脅感
- 按鈕、卡片、邊框統一圓角規格

**避免尖銳**：
- ❌ 直角（0px）
- ❌ 小圓角（4-8px）
- ❌ 尖銳圖形（三角形、星形）
- ✅ 大圓角、圓形、橢圓

### 3. 陰影原則

**擴散式陰影**：
- 使用軟陰影，無硬邊
- 格式：`0 Xpx Ypx rgba(R,G,B,opacity)`
- X值：4-8px（水平擴散）
- Y值：12-20px（垂直擴散）
- 透明度：0.08-0.12（柔和）

**效果**：
- 營造層次感但不過分立體
- 元素"浮"在背景上
- 保持輕盈、通透

### 4. 字型原則

**字型選擇**：
- 使用圓潤的無襯線字型（Rounded Sans-serif）
- 例如：圓體-簡、Quicksand、Varela Round

**字重**：
- 標題：600-700（Medium-Bold）
- 正文：400-500（Regular-Medium）

**行距**：
- 1.7-1.8倍（寬鬆，呼吸感）
- 兒童閱讀需要更多留白

**字間距**：
- 0.02-0.05em（稍寬，易識別）

### 5. 裝飾原則

**裝飾元素**：
- 星星✨、愛心💕、雲朵☁️、花朵🌸
- 氣球🎈、彩虹🌈、蝴蝶🦋

**使用規則**：
- 選擇2-3個元素組合
- 透明度：0.15-0.25（不搶眼）
- 旋轉角度：±15°（ playful但不混亂）
- 散佈在留白區域

**避免**：
- 過度使用（裝飾面積不超過10%）
- 不相關的元素（如：兒童主題不要用齒輪、骷髏）

### 6. 留白原則

**留白比例**：
- 整體留白：55-60%的畫布
- 內容密集度：低
- 元素間距：24-32px（8px的倍數）

**作用**：
- 營造輕鬆、不壓迫的感覺
- 讓重要內容更突出
- 符合兒童認知特點（資訊不過載）

### 7. 互動原則

**按鈕/可點選元素**：
- 明顯的視覺反饋
- Hover狀態：顏色加深10-15%
- 陰影增強：0 8px 20px（更明顯）
- 過渡動畫：200-300ms（smooth）

**鼓勵性文案**：
- "你真棒！"
- "繼續加油！"
- "試試看！"
- "你一定能做到！"

## 視覺層次

### 主要元素（最突出）
- 大標題（字號最大）
- 主要內容（漢字、圖片）
- CTA按鈕（最亮顏色）

### 次要元素
- 副標題、標籤
- 拼音、說明文字
- 裝飾元素

### 輔助元素
- 日期、簽名
- 細節說明
- 背景紋理

## 情緒傳遞

**透過設計傳達**：
- 安全感（柔和、圓潤）
- 好奇心（明亮、有趣）
- 鼓勵（正向、積極）
- 成就感（星星、獎章元素）

## 避免的陷阱

❌ 過於鮮豔（飽和度>80%）
❌ 過於灰暗（明度<60%）
❌ 過於複雜（資訊過載）
❌ 過於立體（ realism不適合）
❌ 尖銳元素（威脅感）
❌ 冷色調（藍色過多會顯得冷淡）

## 示例組合

**典型溫馨可愛風配色**：
```
主色：#FF9AA2（蜜桃粉）
輔色：#B5EAD7（天空藍）
背景：#FFFEF9（奶油白）
文字：#2C2C2C（暖炭灰）
裝飾：✨ opacity:0.2, ☁️ opacity:0.15
```

**典型邊框陰影**：
```
border-radius: 18px
box-shadow: 0 6px 16px rgba(255,154,162,0.10)
```
